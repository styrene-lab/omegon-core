# omegon

Rust-native AI coding agent — lifecycle engine, memory, tool execution.

## Install

```bash
npm install -g omegon
```

This installs a platform-specific prebuilt binary via `@styrene-lab/omegon-{platform}` optional dependencies.

### Alternative install methods

```bash
# Direct binary install (no Node.js required)
curl -fsSL https://omegon.styrene.dev/install.sh | sh
```

## Supported platforms

- macOS ARM64 (Apple Silicon)
- macOS x64 (Intel)
- Linux ARM64
- Linux x64

## Looking for the interactive TUI?

The TypeScript-based interactive TUI harness is a separate package:

```bash
npm install -g omegon-pi
```
