# Omegon Documentation

User-facing documentation for Omegon. Served via [mdserve](https://github.com/jfernandez/mdserve) during development and deployed as part of the release pipeline.

## Local preview

```bash
mdserve docs/
```

## Structure

- [Getting Started](getting-started.md) — installation, first session, configuration
- [CLI Reference](cli-reference.md) — all commands, flags, and subcommands
- [Tools](tools.md) — agent-callable tools and their parameters
- [Architecture](architecture.md) — system overview, data flow, component relationships
- [Configuration](configuration.md) — project profiles, model selection, settings
- [Changelog](../CHANGELOG.md) — release history
